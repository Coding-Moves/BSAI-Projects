# SecureFileStore

A secure cloud-based encrypted file storage system using:
- AES (file encryption)
- RSA (key encryption)
- MongoDB
- Firebase/AWS S3
- JWT Authentication

## Features
✔ Client-side encryption  
✔ Secure key management  
✔ File search  
✔ Cloud-based encrypted storage  
✔ User authentication  

## How to Run

### Backend
```py
cd backend
pip install -r requirements.txt
python run.py
```

### Frontend
```py
cd frontend
npm install
npm start
```
